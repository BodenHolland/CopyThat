# CopyThat

CopyThat is a utility application for macOS that automatically detects two-factor authentication (2FA) codes from incoming SMS messages and allows you to quickly paste them into any browser or application.

## Features
- **Universal Autofill:** Detects 2FA codes from iMessage and makes them available system-wide.
- **Smart Detection:** Uses keyword-based detection for codes.
- **Privacy Focused:** Everything stays local on your machine.
- **Modern UI:** Rebranded with a premium macOS aesthetic.

## How to Build
1. Open `CopyThat.xcodeproj` in Xcode.
2. Select the `CopyThat` scheme.
3. Fix any signing issues in **Signing & Capabilities**.
4. Press **Cmd + R** to run.
